# 깃허브에 프로젝트 올리기 

### 1. 고양이 로고 홈 → new → create repository 
### 2. 프로젝트 우클릭 →  Team(깃과 관련된 모든것들은 여기) →   share project →  check (create repository 눌러줘야 체크가 가능)  → 프로젝트에 원통모양이 생기면 연동되었다는 뜻 
  - 위에 설정 하고 나면 Tema에 다양한 메뉴들이 생김 

### 3. Team  → Commit → Unstaged Change에 있는것들 Staged Changes로 이동 → commit and push (이걸 해야 로컬에 있던걸 git으로 옮겨줌) → 창 뜰때 깃주소 넣고 비밀번호 설정해줘야함 →
