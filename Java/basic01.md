# "Hello Java" ; 이클립스말고 cmd로 출력하기
1. 파일명(클래스명과 동일).java 파일을 만든다 : 소스코딩
2. cd 경로 붙여넣기
3. javac 파일명.java : 컴파일을 한다(컴파일을 하면 .class가 생성됨)
4. java 출력결과 : 클래스를 실행시킨다

- cd ; Change Directory
